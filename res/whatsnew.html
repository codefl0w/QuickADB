What's New?

1) QuickADB has been fully recreated!

- Switched to PyQt6 for scalable, beautiful themes
- Used a lot more threading to achieve maximum smoothness
- Recreated all features and used them as dynamic modules
- Open-sourced the program


2) ADB Debloater is now an App Manager

- Categorize app types by their paths (User, System and Vendor)
- Disable multiple apps at once
- Change app sorting by clicking columns (App status and App type columns only)
- Backup and restore apps with a single click
- Show app details such as version and installation path
- Manage multiple app permissions at once
- Newer, more readable UI
- Improved app detection


3) A new, custom terminal is added

- Based on CMD or your default terminal, all commands work as expected
- Buttons to extract or highlight the output
- Drag-and-drop support for files
- Autocomplete for adb and fastboot commands


4) App-wide enhancements

- A lot of performance enhancements
- Major UI changes - switched from TkInter to PyQt6
- Added the "About" section - the one you're reading right now
- Fixed logging bugs
- Completely modularized the codebase
- Multi-OS support
5) Added themes

- Added 4 premade themes: dark (default), high contrast, light and Android
- Added default theme option: resets the custom palette and inherits OS' design language
