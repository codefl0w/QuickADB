What's New?

1) ADB Debloater is now an App Manager

- Categorize app types by their paths (User, System and Vendor)
- Disable multiple apps at once
- Change app sorting by clicking columns (App status and App type columns only)
- Backup and restore apps with a single click
- Show app details such as version and installation path
- Manage multiple app permissions at once
- Newer, more readable UI
- Improved app detection


2) A new, custom terminal is added

- Based on CMD, all commands work as expected
- Buttons to extract or highlight the output
- Drag-and-drop support for files
- Custom UI instead of the classic boring CMD look
- Pure CMD from older versions is still accessible


3) Script-wide enhancements

- Threading support to run multiple commands at once, much faster UI and real-time logging
- Some UI changes for better readability
- Added the "About" section - the one you're reading right now
- Fixed logging mistakes
- QuickADB now has two variants: full and lite. Lite versions do not have driver installers, making the executable only around 30 megabytes
